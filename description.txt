Причины выбора проекта:
    Приятное времяпрепровождение, досуг.

Цель и задачи проекта:
    Цель: Создать умную колонку на основе raspberry pi.
    Задачи:
        1. Подобрать компоненты
        2. Сделать 3D модель и печатную плату
        3. Написать ПО

Описание продукта (устройства):
    Дрон включает в себя:
        1. микроконтроллер Raspberry pi
        2. детали от умной колонки "Сбер"
        3. процессор STM 32
        4. светочувствительный резистор
        5. светодиоды
    Колонка может отвечать на простые запросы и зачитывать тгк "физтех.confessions" при понижении уровня света

Развернутое описание процесса решения поставленных задач.
1) Для создания 3D моделей использовали SolidWorks, для нарезки слоёв для печати использовали OrcaSlicer.
2) Для программирования и прошивки микроконтроллера использовали официальную документцаию:
    https://docs.espressif.com/projects/esp-idf/en/stable/esp32

Анализ существующих аналогов и формулировки отличительных признаков проекта
Аналоги:
    1. https://sberdevices.ru/sberboom-mini2/
Отличительные признаки:
    1. возможность легко добавить необходимые компоненты (модуль радиосвязи, GPS-трекер)

Описание процесса проектирования и изготовления продукта:
1) Подобраны компоненты и спроектирована модель корпуса
2) Собрана электроника и соеденена с корпусом
3) Написано ПО для взаимодействия с колонкой

На фабрике выполнены части работы:
1) Печать всех компонент корпуса на 3D принтере
2) Пайка компнентов

Вне фабрики выполнено:
1) Подбор компонентов
2) Написание ПО
3) Сборка электроной части колонки
4) Тестировка получившегося изделия

Описание процесса тестирования продукта и анализ его результатов:
1) Тестирование реакции колонки на простые запросы
2) Включение колонки при понижении уровня света
